# 주석 보존 규칙

코드 수정 시 기존 주석이 제거되는 경우, 태그 주석은 반드시 의도를 확인한다.

## 보호 대상: 태그 주석

`[SPEC]`, `[CORE]`, `[TODO]`, `[WARN]` 태그가 붙은 주석은 보호 대상이다.

교체 블록에 태그 주석이 포함되어 있으면:
1. 주석의 의도가 새 코드에서도 여전히 유효한지 확인
2. **유효하면** — 새 코드에 주석을 유지하거나 위치를 조정
3. **무효하면** — 변경 계획에 "제거되는 주석: `[TAG] 내용` — 사유: ..." 를 기재

## 판단 불요: 자동 제거 가능한 주석

- 코드를 그대로 번역한 중계형 주석
- 주석 처리된 코드 (commented-out code)
- 코드와 불일치하는 주석 (주석 가이드 §주석을 달기 전에 #3)
