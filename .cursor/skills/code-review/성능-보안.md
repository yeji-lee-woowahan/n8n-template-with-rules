## 성능 민감 기능

### 코드 스플리팅

```typescript
// 페이지 컴포넌트는 lazy로 불러오기
const McpListPage = lazy(() => import('@pages/mcp/McpListPage'));
```

### Tree Shaking

전체 import 금지, 필요한 것만 import:

```typescript
// Bad - 전체 라이브러리 import
import _ from 'lodash';
import * as Icons from '@atelier/blue/icons';

// Good - 필요한 것만 import
import debounce from 'lodash/debounce';
import { CheckIcon, HomeIcon } from '@atelier/blue/icons';
```

### 가상화 (100개 이상 렌더링 시)

#### Atelier 컴포넌트 (무한스크롤 내장)

**SelectWithSearch** - 옵션이 많을 때:

```tsx
<SelectWithSearch
  fetchNextPage={() => fetchNextPage()}
  hasNextPage={hasNextPage}
  searchValue={search}
  onChangeSearch={setSearch}
>
  {options.map(opt => (
    <Select.Option key={opt.id} value={opt.id}>{opt.name}</Select.Option>
  ))}
</SelectWithSearch>
```

| prop | 설명 |
|------|------|
| `fetchNextPage` | 다음 페이지 로드 함수 |
| `hasNextPage` | 무한스크롤 가능 여부 |

**Table** - ref를 사용해 무한스크롤 직접 구현

#### 일반 리스트 (react-virtuoso)

```typescript
import { Virtuoso } from 'react-virtuoso';

<Virtuoso
  data={items}
  itemContent={(index, item) => <ListItem item={item} />}
/>
```

#### 가상화가 어려운 경우

CSS로 렌더링 최적화:

```css
.list-item {
  content-visibility: auto;
  contain-intrinsic-size: 0 50px; /* 예상 높이 */
}
```

## 보안 민감 기능

### XSS 방어

| 상황 | 대응 |
|------|------|
| `dangerouslySetInnerHTML` 사용 시 | `DOMPurify` 적용 필수 |
| 사용자 입력 링크 (`href`, `src`) | `javascript:` 프로토콜 차단 |

```typescript
import DOMPurify from 'dompurify';

// dangerouslySetInnerHTML 사용 시
<div dangerouslySetInnerHTML={{ __html: DOMPurify.sanitize(htmlContent) }} />

// 링크 검증
const isSafeUrl = (url: string) => {
  try {
    const parsed = new URL(url);
    return ['http:', 'https:'].includes(parsed.protocol);
  } catch {
    return false;
  }
};
```

### 민감 정보 저장

| 금지 | 이유 |
|------|------|
| JWT를 localStorage에 저장 | XSS로 탈취 가능 |
| AccessToken을 localStorage에 저장 | XSS로 탈취 가능 |
| PII(개인식별정보)를 localStorage에 저장 | 보안 위험 |

→ 토큰은 httpOnly 쿠키 또는 메모리에 저장

### 외부 링크

`target="_blank"` 사용 시 `rel` 속성 필수:

```tsx
// Bad
<a href={externalUrl} target="_blank">외부 링크</a>

// Good
<a href={externalUrl} target="_blank" rel="noopener noreferrer">외부 링크</a>
```
